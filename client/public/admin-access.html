Admin Access Helper

This page will help you access the admin panel in GrowWise.

Steps to access the admin panel:

  1. Make sure you're logged in to your account
  2. In the bottom left of the sidebar, click on your profile dropdown (chevron down icon)
  3. Look for the "Admin Panel" option in the dropdown menu
  4. If you don't see it, use the button below to redirect to the admin panel
Note: If you're still having trouble accessing the admin panel, try logging out and logging back in, as your user has been granted admin privileges but it might require a fresh session.
